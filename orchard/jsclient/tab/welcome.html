Orchard is a proof-of-concept daemon and web client for personal ConcurrenTree documents. It aims to promote open concurrency standards and provide P2P Wave-like collaboration in secure, difficult-to-censor ways. It is developed primarily by Philip Horger and Nathaniel Abbots, primarily. The technology is still not stable, but when it is, it will be fairly straightforward to write other utilities and programs to interact with your concurrent data using the MCP protocol.
